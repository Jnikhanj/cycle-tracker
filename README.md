# Cycle Tracker

A personal, local-only period tracking app.

## What it does

- Records period start and end dates
- Tracks daily flow, pain score, mood, symptoms, and notes
- Shows a calendar with recorded periods, predicted periods, and an estimated fertile window
- Calculates average cycle length and average period length
- Exports and imports a local JSON backup
- Stores data only in the browser using localStorage

## Privacy

This app has no account, no server, no external analytics, and no subscription. Cycle data is saved only in the browser/device where the app is used. Export a backup before clearing browser data or changing phone.

## Safety note

This app is for personal tracking only. It does not diagnose medical conditions and should not be used as contraception, fertility treatment advice, or medical advice.
